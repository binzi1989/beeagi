# 桌面端说明（中文）

## 1) 安装与运行

```bash
npm install
npm run dev
```

本地调试桌面壳：

```bash
npm run tauri dev
```

## 2) 可选环境变量

- `VITE_API_BASE_URL`（默认：`http://127.0.0.1:8000`）
- `VITE_CONTROL_PLANE_API_KEY`（后端启用 API Key 时必填）

## 3) 页面结构（当前版本）

- `工作台`：任务输入、对话式时间线、交付产物、蜂群状态
- `LLM 与 Token`：模型路由配置、Token 统计总览、按模型统计、最近任务统计

## 4) 工作台体验设计

- 场景优先：编程 / 办公 / 研究 / 调试 / 数据 / 产品
- 交付优先：左侧主区域聚焦输入与最终结果
- 低压交互：右侧改为标签页（反馈 / 进化 / 系统），减少信息轰炸
- 快速闭环：交付区提供“复制产物 / 一键采纳 / 需要修订”

## 5) 反馈与进化

- 支持人工反馈：评分 + 修正建议 + 隐式信号
- 支持自动反馈：用户忘记反馈时，系统可从对话推断反馈并驱动进化链路
- 候选治理：影子回放、灰度观察、晋升与回滚

## 6) 常见排查

1. 页面请求失败：先检查 `VITE_API_BASE_URL` 与后端端口。  
2. 405/跨域问题：确认后端已启用 CORS 配置并包含 `OPTIONS`。  
3. 无法提交写接口：若后端启用 API Key，需设置 `VITE_CONTROL_PLANE_API_KEY`。
