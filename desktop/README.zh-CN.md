# 桌面端快速说明（中文）

## 1) 安装依赖

```bash
npm install
```

## 2) 启动前端开发服务

```bash
npm run dev
```

## 3) 启动 Tauri 桌面壳

```bash
npm run tauri dev
```

## 可选环境变量

- `VITE_API_BASE_URL`（默认：`http://127.0.0.1:8000`）
- `VITE_CONTROL_PLANE_API_KEY`（后端启用 API Key 时必填）

## 场景化界面（新版）

- 界面已经改成“场景优先”，不是“功能堆叠”。
- 内置场景：
  - 编程场景：实现 -> 评审 -> 修复
  - 通用办公场景：整理 -> 生成 -> 修订
  - 研究分析场景：检索 -> 分析 -> 结论
  - 扩展场景：故障排查、数据处理、产品方案、运营增长
- 支持“一键跑全流程”，也支持单任务手动执行。
