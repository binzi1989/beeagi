# v0.2.0 - Active Scout Loop + Launch-Ready Docs

Release date: 2026-05-17

## Highlights

- Added active Scout pheromone loop for proactive self-optimization:
  - signal deposit from task context
  - evaporation and ttl control
  - top-k pheromone injection into Worker runtime
  - feedback reward/punishment update
  - patrol mode for historical sampling
- Added evolution APIs:
  - `GET /evolution/pheromones`
  - `POST /evolution/scout-patrol`
- Desktop UI now shows Scout pheromone state and supports one-click patrol.
- Reworked English and Chinese README for fast onboarding and open-source visibility.
- Added launch assets:
  - 30-second demo scripts (EN/ZH)
  - GitHub launch playbook (ZH)

## API Changes

### New

- `GET /evolution/pheromones`
  - query params:
    - `limit` (default 100)
    - `intentCluster` (optional)
    - `onlyActive` (default true)
- `POST /evolution/scout-patrol`
  - body:
    - `sampleSize` (default 30)

## Data Model

### New Table

- `scout_pheromones`
  - intent cluster, source route, novelty/reliability/cost
  - strength/reward, ttl, usage/success/failure counts
  - last_seen/expires timestamps and metadata
