# v0.2.0 - 主动 Scout 信息素闭环 + 开源发布素材

发布日期：2026-05-17

## 版本亮点

- 新增主动 Scout 信息素闭环（更主动的自优化）：
  - 从任务上下文沉积信息素
  - 基于时间衰减与 TTL 的蒸发机制
  - 执行阶段注入 Top-K 信息素到 Worker
  - 根据用户反馈对信息素做奖惩更新
  - 支持巡检模式（从历史任务主动采样）
- 新增进化接口：
  - `GET /evolution/pheromones`
  - `POST /evolution/scout-patrol`
- 桌面端新增信息素可视化与“一键巡检”操作。
- 全面重写中英文 README，降低首次上手门槛。
- 新增开源发布素材：
  - 30 秒演示脚本（中/英）
  - GitHub 增长作战手册（中文）

## API 变化

### 新增接口

- `GET /evolution/pheromones`
  - 查询参数：
    - `limit`（默认 100）
    - `intentCluster`（可选）
    - `onlyActive`（默认 true）
- `POST /evolution/scout-patrol`
  - 请求体：
    - `sampleSize`（默认 30）

## 数据模型

### 新增表

- `scout_pheromones`
  - 意图簇、来源路线、novelty/reliability/cost
  - strength/reward、ttl、usage/success/failure
  - last_seen/expires 时间与 metadata

## 迁移说明

- 项目启动时通过 `Base.metadata.create_all` 自动创建 `scout_pheromones`。
- 本地 SQLite 无需额外手工迁移。

## 兼容性

- 无破坏性移除。
- 既有任务/反馈/进化链路保持兼容。
