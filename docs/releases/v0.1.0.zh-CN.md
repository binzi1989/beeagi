# BeeAGI v0.1.0 发布说明（中文）

发布日期：2026-05-17

## 本版重点

- 四角色蜂群链路已可端到端运行：
  斥候 Scout -> 工蜂 Worker -> 蠕虫 Worm -> 蜂王 Queen
- 桌面端体验聚焦“结果交付”：
  对话式时间线 + 交付产物主舞台
- 进化治理具备上线能力：
  影子回放、5% 灰度、晋升、回滚、审计链路
- LLM 运维可视化：
  运行时配置页面 + Token 统计页面

## 关键能力

- 任务与反馈：
  - `POST /tasks`
  - `GET /tasks/{id}`
  - `POST /tasks/{id}/feedback`
  - `POST /tasks/{id}/auto-feedback`
- 技能与进化：
  - `POST /skills/{id}/candidate`
  - `POST /skills/{id}/promote`
  - `POST /skills/{id}/rollback`
  - `POST /skills/{id}/candidate/{candidate_id}/shadow-replay`
  - `GET /skills/{id}/candidate/{candidate_id}/canary-status`
  - `POST /evolution/auto-promote`
  - `GET /evolution/events`
  - `GET /evolution/candidate-audits`
  - `GET /evolution/hardening-report`
- LLM：
  - `GET /llm/config`
  - `PUT /llm/config`
  - `GET /llm/token-stats`

## 质量验证

- 后端测试：通过
- 桌面端构建：通过

## 运营建议

1. 非本地环境务必启用 `APP_CONTROL_PLANE_API_KEY`。  
2. 严禁提交 `.env`、密钥、数据库文件。  
3. 变更上线优先走“影子回放 -> 灰度 -> 晋升/回滚”流程。
